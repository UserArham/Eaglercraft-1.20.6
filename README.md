# Eaglercraft 1.20.6 

A port of Eaglercraft 1.20.6, well it took me 4 months to make it though. This has a lighter 13.5 MB `assets.epk` file compared to Chromecraft's 14.2 MB file.


The size of `assets.epk` is the equivalent to the `classes.js` file since it is only 13.5 MB only.

This added bees, cherry stuff like cherry leaves, netherite, and wardens. This is **NOT** based on 1.12.2 Eaglercraft code. This is based on 1.8.8. As you remember 1.8.8 is a legacy build of Minecraft in 2015 and ported in 2024 on a browser.

## What it requires

It requires:

- 4GB or more RAM
- 16 GB storage or more
- TLS 1.2 or higher
- Chrome 68 or higher
- ChromeOS, Mac, Windows, Linux, and Ubuntu

## What this supports

For Android, it supports:

- Android 7.1 or higher with Chrome 68

For iOS, it supports:

- iOS 12 or higher with Chrome 68

## Browsers needed:

- Chrome 68+
- Edge 70+
- Firefox 71+
- Opera 69
- Safari 47+
